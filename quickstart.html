クイックスタート

3ステップで始めよう

1

インストール

npmを使ってグローバルにインストールします。

terminal
$ npm install -g @anthropic-ai/claude-code
必要条件: Node.js 18以上
2

プロジェクトに移動

作業したいプロジェクトのディレクトリに移動します。

terminal
$ cd /path/to/your-project
Tip: Gitリポジトリ内で実行すると、Git機能もフル活用できます
3

起動

claudeコマンドを実行して対話を開始します。

terminal
$ claude
✓ Claude Code ready. How can I help?
初回起動時: Anthropic APIキーの設定が必要です

便利なコマンド

覚えておくと便利なスラッシュコマンド

コマンド
説明
/help
ヘルプを表示
/clear
会話履歴をクリア
/compact
会話を要約してコンテキストを圧縮
/cost
現在のセッションのAPI使用量を表示
/doctor
Claude Codeの設定を診断
/init
プロジェクト用のCLAUDE.mdを生成

Tips

より効果的に使うためのヒント

📁

CLAUDE.mdを活用

プロジェクトルートにCLAUDE.mdファイルを置くと、プロジェクト固有のコンテキストをClaudeに伝えられます。

🎯

具体的に指示

「修正して」よりも「src/auth.tsの validateToken関数でトークンの有効期限チェックを追加して」のように具体的に。

📋

タスクを分割

大きな機能追加は複数の小さなタスクに分割すると、より正確に実装できます。

🔄

フィードバックを与える

結果を確認して「ここを直して」「これでOK」などフィードバックを与えると、より良い結果に。
