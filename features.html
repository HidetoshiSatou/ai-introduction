機能一覧

Claude Codeができること

📝

コード編集・作成

ファイルの読み取り、編集、新規作成を行います。既存コードを理解した上で、適切な修正を提案・実行します。

  • 既存コードの修正・リファクタリング
  • 新機能の実装
  • バグ修正
  • コードレビュー・改善提案
🔍

コードベース探索

プロジェクト全体を検索・分析し、コードの構造や依存関係を把握します。

  • ファイル検索（Glob パターン）
  • コード内テキスト検索（Grep）
  • コード構造の理解
  • 依存関係の分析
🔧

Bash コマンド実行

ターミナルコマンドを実行し、開発ワークフローを自動化します。

  • ビルド・テストの実行
  • パッケージ管理（npm, pip, cargo等）
  • Docker操作
  • システムコマンド
📊

Git 操作

バージョン管理をサポートし、コミットやPR作成を効率化します。

  • コミット作成（適切なメッセージ付き）
  • ブランチ管理
  • プルリクエスト作成
  • 差分確認・マージ
🌐

Web 検索・情報取得

最新情報を取得して、ドキュメントやライブラリの情報を活用します。

  • Web検索で最新情報を取得
  • WebページのコンテンツをFetch
  • 公式ドキュメント参照
  • API情報の調査
📋

タスク管理

複雑なタスクを整理・追跡し、進捗を可視化します。

  • ToDoリスト作成・管理
  • 進捗状況の追跡
  • タスクの分解・計画
  • マルチステップ作業の管理
🤝

並列処理

複数のタスクを同時に実行し、作業効率を最大化します。

  • 複数ファイルの同時読み取り
  • 並列検索の実行
  • バックグラウンドタスク
  • エージェントの並列起動
📓

Jupyter Notebook

データ分析ノートブックの編集をサポートします。

  • セルの追加・編集・削除
  • コード・マークダウンセル
  • 出力の確認
  • データ分析サポート
💡

インタラクティブ支援

対話的に開発をサポートし、最適な実装方針を一緒に検討します。

  • 質問への回答・説明
  • 選択肢の提示
  • 実装方針の相談
  • プランニングモード
